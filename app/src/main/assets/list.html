Adding an item

Enter text into the box shown and press enter to add an item

Change the category you add items to using the dropdown next to your input
Read more on categories

Note: You cannot add two of the same item to the same category

Using the list

Check off an item by tapping on it

Note: If there is no shop set when an item is checked then it will not be saved to history
See: Shops

Delete an item by swiping it

Uncheck an item by tapping on it. It will return to it's original category.
The category is shown next to the return icon

Clear and save items to history from Cleared using the clear button at the top of the section

Categories

Collapse a category by tapping it

Add a category using the dropdown menu and selecting Add new...

Note: Categories will be deleted when you move off the page if there are no items in them

Shops

Set the shop by pressing Set shop in the top bar

Clear the shop by pressing Clear shop when a shop is active

Note: Shops are automatically cleared a day after it was set or when you are far away (150m)

Setting a reminder

You can set a reminder to go shopping using the bell in the action bar

Note: You can only have one reminder at a time, setting a new one will clear the current reminder
