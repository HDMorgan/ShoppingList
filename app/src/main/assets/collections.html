Overview

Collections are collections of items that can be added to your main list. For example, your weekly shopping

Creating a collection

Press the plus button in the action bar to create a new collection

Adding to the main list

Press the plus symbol on a collection to add it's items to your main list

Editing a collection

Click on a collection to edit it. Editing items works exactly the same as the main list
See: Main list

Delete the collection by pressing the bin icon in the action bar
